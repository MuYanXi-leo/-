# 云购优选移动端电商前端项目

这是一个可直接运行的移动端类电商系统前端项目，页面使用 HTML5、CSS3、JavaScript 实现。

## 页面清单

1. 首页：index.html
2. 分类页：category.html
3. 商品列表页：list.html
4. 商品详情页：detail.html
5. 购物车页：cart.html
6. 登录页：login.html
7. 注册页：register.html
8. 我的页面：user.html
9. 订单确认页：order.html
10. 地址管理页：address.html

## 运行方式

直接双击 index.html 即可运行。建议使用 VS Code 的 Live Server 插件运行，并在浏览器开发者工具中切换为手机尺寸查看。

## 图片说明

项目中的商品图和首页 Banner 使用网上真实图片链接，演示时需要联网。assets 文件夹中保留了本地 SVG 兜底图，若网络图片加载失败，页面会自动显示兜底图。

## 主要技术

- HTML5 语义化页面结构
- CSS3 Flex 布局、Grid 布局、移动端适配
- JavaScript DOM 渲染、页面跳转、购物车数量计算、表单校验
- 多页面移动端电商业务流程：首页 → 列表 → 详情 → 购物车 → 订单 → 我的
